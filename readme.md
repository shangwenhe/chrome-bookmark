全屏瀑布流导航页 (Vimium 版)

一个功能丰富的 Chrome 新标签页扩展，提供书签和历史记录的瀑布流展示，支持 Vimium 风格的快捷键操作。
功能特性
瀑布流布局: 书签和历史记录采用瀑布流布局，美观且易于浏览
Vimium 快捷键: 支持 Vimium 风格的快捷键操作
按 f 键打开链接提示
按 j/k 键上下滚动
按 / 键打开搜索
历史记录分组: 历史记录按日期分组展示（X月X日 X时）
智能搜索: 支持关键词搜索书签和历史记录
自动加载: 滚动到底部时自动加载更多内容
响应式设计: 适配不同屏幕尺寸
安装方法

1. 下载项目文件
2. 在 Chrome 中打开 chrome://extensions/
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
使用说明
打开新标签页即可使用
按 / 键搜索书签或历史记录
按 f 键查看链接提示，输入提示字符访问链接
滚动到底部自动加载更多内容
